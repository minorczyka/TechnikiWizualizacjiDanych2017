Wykorzystane przeze mnie dane pochodz z Rankingu Dzielnic Otodom, ktrego autorem jest serwis z ogoszeniami nieruchomoci Otodom (https://s3-eu-west-1.amazonaws.com/landingi-editor-uploads/Wtc0cc7t/OTODOM_Ranking_Krakow_20170915_web.pdf). W badaniu przeprowadzonym na przeomie wrzenia i padziernika 2016 Otodom wsplnie z OLX i pod nadzorem agencji badawczej PBS, zapyta ponad 140 000 Polek i Polakw o ocen rnych aspektw ycia w miejscu ich zamieszkania.

df <- read.csv("dzielnice.csv")
df_scale <- data.frame(dzielnica = df$dzielnica, df[-1] %>% apply(2, function(x) (x - min(x)) / diff(range(x))))
df_m <- melt(df_scale) %>%
  arrange(variable, dzielnica) %>% 
  mutate(variable = recode(variable, 
                           czysta.okolica = "czysta okolica",
                           zadbana.okolica = "zadbana okolica",
                           atwy.dojazd = "atwy dojazd",
                           czas.wolny = "czas wolny",
                           wygodne.zakupy = "wygodne zakupy",
                           godne.polecenia = "godne polecenia"))
## Using dzielnica as id variables

W rankingu przestawione s wyniki dla kadej z dzielnic w poszczeglnych kategoriach oraz oglne podsumowanie, w ktrym dla kadej kategorii przedstawiono trzy najlepsze dzielnice. Brakuje jednak zbiorczego podsumowania, w ktrym mona dla konkretnej dzielnicy okreli, w ktrych kategoriach wypada sabo, a w ktrych dobrze. Umoliwia to natomiast stworzony przeze mnie wykres, dodatkowo atwo zauway, e szczeglnie korzystanie w rankingu wypady dzielnice takie jak Zwierzyniec, Czyyny czy Bronowice, natomiast wyjtkowo sabo Swoszowice i Nowa Huta.

ggplot(df_m, aes(variable, dzielnica)) +
  geom_tile(aes(fill = value)) +
  scale_fill_gradient2(low = "red", mid = "yellow", high = "green4", 
                       midpoint = 0.5, breaks=c(0, 1), labels=c("najgorszy", "najlepszy")) +
  labs(x = "Kategoria", y = "Nazwa dzielnicy", fill = "Wynik", title = "Ranking dzielnic Krakowa") +
  theme(axis.text.x = element_text(angle = 315, hjust = 0))
